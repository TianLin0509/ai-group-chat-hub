Codex 设置

全 Hub 新建 Codex 会话统一生效。

订阅 / Claude Code 登录 使用当前本机 Claude Code 登录状态。Hub 会保持现有订阅启动方式，不写入 API Key。
API mode sets ANTHROPIC_BASE_URL / ANTHROPIC_AUTH_TOKEN and clears proxy env so Claude Code connects directly to the configured endpoint.
订阅 / Gemini CLI 登录 使用当前本机 Gemini CLI 登录状态。API 模式暂不接入，避免影响当前可用链路。
API 模式使用隔离 CODEX_HOME；执行权限仍遵循上方“AI 执行权限”，默认不会绕过审批与沙箱。
API 模式 DeepSeek 当前通过 Anthropic 兼容 API 接入，使用隔离 Claude 配置目录。
